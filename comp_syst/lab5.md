# Лабораторная работа №5. Тестирование качества монитора

**Цель:** диагностика монитора и выбор оптимальных параметров цветопередачи.

---

## 1. Характеристики монитора (пример)

| Параметр | Значение |
|----------|----------|
| **Модель** | Dell P2219H |
| **Разрешение** | 1920×1080 (Full HD) |
| **Соотношение сторон** | 16:9 |
| **Разъем подключения** | DisplayPort |
| **Диагональ** | 21.5" |
| **Цветов** | 16.7 млн |
| **Плотность пикселей** | 102 ppi |

**Назначение тестов:**
- **Scale** — плавность градиентов, равномерность заливки, поиск битых пикселей
- **Gamma** — точность передачи полутонов (гамма-кривая)
- **Contrast** — различимость деталей в тенях и на свету

---

## 2. Результаты тестирования

### 2.1 Scale (градиенты)
- Переходы плавные, незначительное **полосение** на сложных градиентах
- Битых пикселей **нет**
- При изменении угла обзора: вертикаль — инверсия цветов, горизонталь — незначительные искажения

### 2.2 Gamma (гамма-кривая)
*Данные для белого цвета:*

| Расстояние, мм | Gamma |
|----------------|-------|
| 40 | 0.89 |
| 80 | 0.95 |
| 120 | 1.01 |
| 160 | 1.08 |
| 200 | 1.16 |
| 240 | 1.35 |
| 280 | 1.46 |
| 320 | 1.58 |
| 360 | 1.72 |

**Вывод:** оптимальное слияние полос при **γ = 1.35–1.46** — полутона слегка завышены.

### 2.3 Contrast
- **Порог различимости:** 2–4% (черный), 96–98% (белый)
- **Оптимальный шаг насыщенности:** **20%** (рекомендация)

### 2.4 Pixel Persistence (шлейфы)
- Длина шлейфа при средней скорости: **2–3 мм**
- Инерционность средняя, для динамичных игр может быть недостаточной

### 2.5 Printer Test
- **Слияние с белым:** 1–3% яркости
- **Слияние с черным:** 1–4% яркости

## 3. Контрольные вопросы

### 3.1 Виды мониторов
**По технологии:**
- *ЭЛТ (CRT)* — устаревшие, громоздкие
- *ЖК (LCD)* — TN (быстрый, плохие углы), IPS (цвет/углы), VA (глубокий черный)
- *OLED* — идеальный черный, беск. контраст
- *Плазма* — для ТВ, в ПК не используется

**По назначению:** офисные, для дизайна, игровые, профессиональные

**По покрытию:** матовые, глянцевые

### 3.2 Конструкция ЖК-монитора
1. Подсветка (LED/CCFL)
2. Рассеиватель + отражатель
3. Поляризатор №1
4. TFT-подложка (управление)
5. Жидкие кристаллы (LC)
6. Цветовой фильтр (RGB)
7. Поляризатор №2
8. Защитное покрытие
