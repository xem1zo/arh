# Основные команды Bash для Linux

## 📂 Навигация (Navigation)

| Команда | Описание |
|---------|----------|
| `pwd` | Показать текущую рабочую директорию (print working directory) |
| `cd ~` | Перейти в домашнюю директорию |
| `cd /путь/к/папке` | Перейти в указанную директорию |
| `cd "Папка с пробелами"` | Переход в папку с пробелами в названии (обязательно в кавычках) |
| `cd ..` | Подняться на один уровень вверх (родительская директория) |
| `cd .` | Остаться в текущей директории (полезно для некоторых операций) |
| `cd -` | Вернуться в предыдущую директорию |
| `ls` | Показать содержимое текущей директории |
| `ls -a` | Показать все файлы, включая скрытые (начинающиеся с `.`) |
| `ls -l` | Показать подробный список с правами, владельцем, размером и датой |
| `ls -la` | Комбинация `-l` и `-a` — подробный список со скрытыми файлами |
| `ls ~/Documents` | Показать содержимое указанной директории |

---

## 📁 Работа с файлами и директориями

| Команда | Описание |
|---------|----------|
| `touch имя_файла` | Создать пустой файл |
| `touch ../../file.txt` | Создать файл на две папки выше текущей |
| `mkdir имя_папки` | Создать новую директорию |
| `mkdir -p путь/к/папке/вложенная` | Создать вложенную структуру папок (создаёт все промежуточные) |
| `mkdir ~/my-project` | Создать папку в домашней директории |
| `cp что_копируем куда` | Скопировать файл |
| `cp file.txt backup/` | Скопировать файл в папку backup |
| `cp file1.txt file2.txt file3.txt target/` | Скопировать несколько файлов в одну папку |
| `cp -r папка_источник папка_назначение` | Рекурсивно скопировать папку со всем содержимым |
| `mv что_перемещаем куда` | Переместить файл или папку (или переименовать) |
| `mv file.txt newname.txt` | Переименовать файл |
| `mv file.txt ../` | Переместить файл на уровень выше |
| `rm имя_файла` | Удалить файл (безвозвратно!) |
| `rm -r имя_папки` | Рекурсивно удалить папку со всем содержимым |
| `rm -rf имя_папки` | Принудительно удалить папку (опасно, не спрашивает подтверждения) |
| `rmdir имя_папки` | Удалить пустую папку |
| `rmdir -p путь/к/папке` | Удалить вложенные пустые папки |

## 📄 Просмотр содержимого файлов

| Команда | Описание |
|---------|----------|
| `cat имя_файла` | Вывести содержимое файла в терминал (только для текстовых) |
| `cat file1.txt file2.txt` | Объединить и вывести содержимое нескольких файлов |
| `less имя_файла` | Постраничный просмотр большого файла (выход — `q`) |
| `head имя_файла` | Показать первые 10 строк файла |
| `head -n 20 имя_файла` | Показать первые 20 строк |
| `tail имя_файла` | Показать последние 10 строк файла |
| `tail -f имя_файла` | Следить за файлом в реальном времени (логи) |
| `wc имя_файла` | Подсчитать строки, слова и символы в файле |
| `wc -l имя_файла` | Подсчитать только количество строк |

## 🔍 Поиск и фильтрация

| Команда | Описание |
|---------|----------|
| `grep "текст" имя_файла` | Найти строки с указанным текстом в файле |
| `grep -r "текст" папка/` | Рекурсивно искать текст во всех файлах в папке |
| `grep -i "текст" файл` | Искать без учёта регистра |
| `find /путь -name "имя*"` | Найти файлы по имени (с поддержкой масок) |
| `find . -type f -size +10M` | Найти файлы больше 10 МБ в текущей папке |
| `which команда` | Показать путь к исполняемому файлу команды |
| `whereis команда` | Показать все пути, связанные с командой |
| `history` | Показать историю введённых команд |

---

## 🔐 Права доступа и владелец

| Команда | Описание |
|---------|----------|
| `chmod 755 имя_файла` | Установить права: владелец — чтение/запись/исполнение, группа и остальные — чтение/исполнение |
| `chmod +x имя_файла` | Сделать файл исполняемым |
| `chmod -x имя_файла` | Убрать права на исполнение |
| `chown владелец:группа имя_файла` | Сменить владельца и группу файла |
| `chown пользователь имя_файла` | Сменить только владельца |
| `chgrp группа имя_файла` | Сменить только группу |
| `ls -l` | Посмотреть текущие права и владельца |

## 📦 Архивация и сжатие

| Команда | Описание |
|---------|----------|
| `tar -cf archive.tar папка/` | Создать tar-архив из папки |
| `tar -xf archive.tar` | Распаковать tar-архив |
| `tar -czf archive.tar.gz папка/` | Создать сжатый архив (gzip) |
| `tar -xzf archive.tar.gz` | Распаковать сжатый архив |
| `zip -r archive.zip папка/` | Создать zip-архив |
| `unzip archive.zip` | Распаковать zip-архив |
| `gzip файл` | Сжать файл в формат .gz |
| `gunzip файл.gz` | Распаковать .gz файл |

## 🌐 Сеть (Network)

| Команда | Описание |
|---------|----------|
| `ping google.com` | Проверить доступность хоста (Ctrl+C для остановки) |
| `ping -c 4 google.com` | Отправить 4 пакета и остановиться |
| `curl http://example.com` | Отправить HTTP-запрос и вывести ответ |
| `curl -O http://example.com/file.zip` | Скачать файл с сохранением имени |
| `wget http://example.com/file.zip` | Скачать файл (сохраняет в текущую папку) |
| `ifconfig` или `ip a` | Показать сетевые интерфейсы и IP-адреса |
| `netstat -tuln` | Показать открытые порты и сетевые соединения |
| `ss -tuln` | Альтернатива netstat (быстрее и современнее) |

## ⚙️ Процессы и система

| Команда | Описание |
|---------|----------|
| `ps` | Показать текущие процессы |
| `ps aux` | Показать все процессы в системе (подробно) |
| `top` | Интерактивный мониторинг процессов (выход — `q`) |
| `htop` | Альтернатива top с более удобным интерфейсом (установить отдельно) |
| `kill PID` | Остановить процесс по его ID |
| `kill -9 PID` | Принудительно остановить процесс (SIGKILL) |
| `pkill имя_процесса` | Остановить процесс по имени |
| `jobs` | Показать фоновые задачи в текущей сессии |
| `bg` | Запустить приостановленную задачу в фоне |
| `fg` | Вернуть фоновую задачу на передний план |
| `&` в конце команды | Запустить команду в фоне (например, `firefox &`) |
| `nohup команда &` | Запустить команду, которая не остановится при закрытии терминала |

## 🧹 Очистка и управление диском

| Команда | Описание |
|---------|----------|
| `clear` или `Ctrl+L` | Очистить экран терминала |
| `df -h` | Показать свободное место на дисках (в человекочитаемом виде) |
| `du -sh папка/` | Показать общий размер папки |
| `du -sh *` | Показать размер всех файлов и папок в текущей директории |
| `free -h` | Показать использование оперативной памяти |

## ❓ Справка

| Команда | Описание |
|---------|----------|
| `man команда` | Открыть руководство по команде (quit — `q`) |
| `команда --help` | Показать краткую справку по команде |

## ⚡ Сочетания клавиш (Terminal Shortcuts)

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+C` | Прервать текущую команду/процесс |
| `Ctrl+Z` | Приостановить текущую команду (поставить на паузу) |
| `Ctrl+D` | Выход из терминала / завершение ввода (EOF) |
| `Ctrl+L` | Очистить экран |
| `Ctrl+A` | Перейти в начало строки |
| `Ctrl+E` | Перейти в конец строки |
| `Ctrl+U` | Удалить всё от курсора до начала строки |
| `Ctrl+K` | Удалить всё от курсора до конца строки |
| `Ctrl+W` | Удалить слово перед курсором |
| `Tab` | Автодополнение имени файла/команды |
| `↑` / `↓` | Пролистывать историю команд |
| `!!` | Повторить последнюю команду |
| `!номер` | Выполнить команду с указанным номером из истории |

---

> ⚠️  Команды `rm`, `rm -r`, `rm -rf` удаляют файлы **безвозвратно** (не в корзину). Используйте их с осторожностью, особенно с `-rf`.
