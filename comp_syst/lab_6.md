# Отчет по лабораторной работе №6
## Тема: Диагностика и тестирование видеоадаптера

### 1. Цель работы
Освоить методику тестирования видеоадаптера с помощью специализированного программного обеспечения (GPU-Z, FurMark, FluidMark). Научиться определять технические характеристики видеокарты и оценивать ее производительность и стабильность в различных режимах нагрузки.

### 2. Индивидуальное задание
Провести комплексное тестирование видеоадаптера:
1.  Определить технические характеристики с помощью GPU-Z.
2.  Выполнить стресс-тест производительности в FurMark.
3.  Выполнить тест физического движка FluidMark.
4.  Зафиксировать показатели FPS и температурные режимы.
5.  Сделать вывод о качестве и производительности видеоадаптера.

### 4. Ответы на контрольные вопросы

#### 1. Какие другие программы используют для тестирования видеокарт?
Помимо использованных GPU-Z и FurMark, для тестирования видеокарт применяются следующие программы:
*   **3DMark:** Самый популярный бенчмарк для сравнения производительности в играх (имеет тесты для разных разрешений и технологий — Time Spy, Fire Strike, Port Royal).
*   **Unigine Heaven / Superposition:** Тесты для оценки стабильности и производительности в сложных графических сценах с поддержкой DirectX и OpenGL.
*   **MSI Afterburner / EVGA Precision X1:** Утилиты для мониторинга параметров (температура, частоты, напряжение) и разгона видеокарты.
*   **OCCT:** Программа для комплексного тестирования компонентов ПК, включающая мощный стресс-тест GPU, который помогает быстро выявить ошибки памяти или перегрев.
*   **Valley Benchmark:** Красивый тест на основе движка Unigine, позволяющий оценить производительность в ландшафтах с большим количеством деталей.

#### 2. Какие программы используют для тестирования мониторов?
Для тестирования мониторов (проверка на битые пиксели, равномерность подсветки, цветопередачу, время отклика) используют специализированные утилиты и онлайн-сервисы:
*   **Nokia Monitor Test:** Классическая портативная программа для проверки геометрии, контрастности, фокусировки и читаемости текста.
*   **EIZO Monitor Test (Monitest):** Набор тестов для оценки цветопередачи, градиентов, резкости и углов обзора.
*   **Dead Pixel Tester:** Простая утилита, которая поочередно заливает экран разными цветами, что позволяет легко обнаружить битые (не меняющие цвет) пиксели.
*   **Online-сервисы (например, tft.vanity.dk):** Сайты, предлагающие набор статических и динамических тестов прямо в браузере для быстрой проверки монитора.

### 5. Выводы
В ходе выполнения лабораторной работы было проведено комплексное тестирование видеоадаптера **Intel(R) HD Graphics 2000**.

*   **Характеристики:** Это интегрированное графическое ядро (встроенное в процессор), работающее на частоте 1100 МГц.
*   **Производительность (FurMark):** В стресс-тесте видеокарта показала очень низкую производительность — всего **2 FPS** в среднем при разрешении 1280x720. Температура GPU составила всего 54°C, что говорит о низкой нагрузке на чип (он просто не способен выдать больше кадров). Частота памяти зафиксирована на уровне 666 МГц.
*   **Физические расчеты (FluidMark):** В тесте физики средний FPS составил **3** при количестве частиц почти 120 тысяч. Тест выполнялся 5 минут и 7 секунд. Физические расчеты шли через центральный процессор (PhysX CPU), так как данная видеокарта не имеет аппаратного ускорения физики.
*   **Общий вывод:** Установленный видеоадаптер Intel HD Graphics 2000 является устаревшим интегрированным решением. Он показывает крайне низкую производительность (2-4 FPS) даже в тестовых приложениях. Данная видеокарта **не пригодна для современных игр** и может использоваться только для базовых офисных задач, просмотра видео (с ограничениями) и работы с 2D-графикой. Состояние карты стабильное (температуры низкие), но ее производительность находится на минимальном уровне.
